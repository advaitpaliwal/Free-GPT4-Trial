# Free-GPT4-Trial

A simple brute force attack for free GPT4 Trials. For educational purposes only.

## Usage

```
git clone https://github.com/advaitpaliwal/Free-GPT4-Trial.git
cd Free-GPT4-Trial
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
